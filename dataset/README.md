# Датасет русской поэзии (all.xml)

## Описание

Файл `all.xml` содержит коллекцию русских стихотворений различных авторов с метаданными.

## Структура XML

### Корневой элемент

```xml
<items>
  <!-- Все стихотворения -->
</items>
```

### Структура каждого стихотворения

Каждое стихотворение обёрнуто в тег `<item>` и содержит 6 основных тегов:

#### 1. `<author>` - Автор стихотворения

**Обязательность:** Да  
**Тип:** Строка

**Примеры:**
```xml
<author>Михаил Лермонтов</author>
<author>Сергей Есенин</author>
<author>Игорь Северянин</author>
```

#### 2. `<text>` - Текст стихотворения

**Обязательность:** Да  
**Тип:** Текст (многострочный)

**Описание:** Полный текст стихотворения со всеми строфами

**Пример:**
```xml
<text>Забывши волнения жизни мятежной,
Я вижу пред собой
Как будто бы берег реки безнадежной,
С увядшею травой...</text>
```

#### 3. `<name>` - Название стихотворения

**Обязательность:** Да  
**Тип:** Строка

**Примеры:**
```xml
<name>Забывши волнения жизни мятежной...</name>
<name>Нивы сжаты, рощи голы...</name>
<name>ЧАРЫ ЛЮЧИНЬ</name>
```

#### 4. `<date_from>` - Год начала написания

**Обязательность:** Нет (может быть пустым)  
**Тип:** Число (год)

**Примеры:**
```xml
<date_from>1829</date_from>
<date_from>1917</date_from>
<date_from></date_from>  <!-- пустое значение -->
```

#### 5. `<date_to>` - Год окончания написания

**Обязательность:** Нет (может быть пустым)  
**Тип:** Число (год)

**Примеры:**
```xml
<date_to>1829</date_to>
<date_to>1918</date_to>
<date_to>1940</date_to>
<date_to></date_to>  <!-- пустое значение -->
```

#### 6. `<themes>` - Темы стихотворения

**Обязательность:** Нет (может быть пустым)  
**Тип:** Контейнер с вложенными `<item>` элементами

**Примеры:**

Пустая тема (наиболее распространённый случай):
```xml
<themes></themes>
```

С одной темой:
```xml
<themes>
  <item>О любви</item>
</themes>
```

## Пример полной записи

### С темой:
```xml
<item>
  <themes>
    <item>О любви</item>
  </themes>
  <author>Лев Мей</author>
  <date_from>1862</date_from>
  <text>Итак, вы ждете от меня
Письма по-русски для науки?...</text>
  <name>Итак, вы ждете от меня...</name>
  <date_to>1862</date_to>
</item>
```

### Без темы:
```xml
<item>
  <themes></themes>
  <author>Сергей Есенин</author>
  <date_from>1917</date_from>
  <text>Нивы сжаты, рощи голы,
От воды туман и сырость...</text>
  <name>Нивы сжаты, рощи голы...</name>
  <date_to>1917</date_to>
</item>
```

## Особенности данных

1. **Порядок тегов:** Теги могут идти в произвольном порядке внутри `<item>`
2. **Пустые значения:** Теги `<date_from>`, `<date_to>` и `<themes>` могут быть пустыми
3. **Темы:** В большинстве случаев тег `<themes>` пустой
4. **Даты:** Даты представлены только годами (4-значные числа)

## Использование

Этот датасет может быть использован для:
- Анализа стиля и эволюции русской поэзии
- Изучения архаизации языка в поэтических текстах
- Обучения моделей машинного обучения для генерации поэзии
- Исследования тематики и периодизации творчества различных поэтов
