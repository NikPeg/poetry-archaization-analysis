# poetry-archaization-analysis

Корпусное исследование архаизации русской поэтической лексики (1820-1980). Анализ частотности устаревших слов и поэтизмов, построение кривых "вымирания", выявление периодов интенсивной архаизации. Материал: 17 тысяч стихотворений, 195 авторов.

## Цель исследования

Проанализировать динамику использования архаичной и устаревшей лексики в русской поэзии XIX-XX веков.

## План исследования

### 1. Подготовка данных (1-2 дня)
- Парсинг XML-датасета стихотворений
- Извлечение текстов и метаданных (автор, год написания)
- Токенизация и очистка текстов
- Формирование выборки по временным периодам

### 2. Формирование словаря архаизмов (1 день)
- Составление базового списка архаичных слов и поэтизмов
- Категоризация по типам (устаревшие слова, старославянизмы, историзмы)

### 3. Анализ частотности (2-3 дня)
- Подсчёт употреблений архаизмов по периодам (по десятилетиям)
- Вычисление относительной частотности (на 1000 слов)
- Построение графиков динамики

### 4. Сравнительный анализ (1-2 дня)
- Сравнение использования архаизмов у разных авторов
- Выявление периодов интенсивной архаизации
- Анализ корреляции с литературными течениями

### 5. Визуализация и выводы (1 день)
- Создание итоговых графиков и таблиц
- Формулирование выводов
- Подготовка краткого отчёта

## Ожидаемые результаты

- Графики динамики использования архаизмов по периодам
- Список авторов с наибольшей/наименьшей архаизацией
- Выявление ключевых периодов изменения поэтического языка
- Статистический отчёт по категориям архаизмов
