# Корпусный анализ архаичной лексики в русской поэзии XVIII–XX веков

Корпусное исследование динамики использования архаизмов в русской поэзии (1720-1990). Анализ частотности устаревших слов и поэтизмов, построение кривых "вымирания", выявление периодов интенсивной архаизации. Материал: 16,694 стихотворения, 195 авторов, 1,726,105 словоупотреблений.

## Цель исследования

Проанализировать динамику использования архаичной и устаревшей лексики в русской поэзии XVIII–XX веков на материале корпуса из 16,694 стихотворений, выявить периоды интенсивной архаизации и установить корреляцию с литературными течениями.

## План исследования

### 1. Подготовка данных ✅
- Парсинг XML-датасета стихотворений
- Извлечение текстов и метаданных (автор, год написания)
- Токенизация и очистка текстов
- Формирование выборки по временным периодам

**Скрипты:**
- `src/parse_xml_to_csv.py` - конвертация XML → CSV/Parquet
- `src/example_load_data.py` - примеры работы с данными

**Результат:** 16,694 стихотворений, 195 авторов (1725-1996)

---

### 2. Формирование словаря архаизмов ✅
- Составление базового списка архаичных слов и поэтизмов
- Категоризация по типам (устаревшие слова, старославянизмы, историзмы)

**Скрипты:**
- `src/parse_archaisms.py` - парсинг словаря устаревших слов
- `src/example_find_archaisms.py` - примеры поиска архаизмов

**Результат:** 1,188 уникальных архаизмов из словаря Азбука веры

---

### 3. Анализ частотности ✅
- Подсчёт употреблений архаизмов по периодам (по десятилетиям)
- Вычисление относительной частотности (на 1000 слов)
- Построение графиков динамики

**Скрипты:**
- `src/word_frequency_analysis.py` - облака слов по векам
- `src/analyze_archaism_frequency.py` - полный анализ частотности
- `src/add_archaisms_from_manual.py` - добавление архаизмов из ручного списка

**Результаты:**
- Облака слов для 18, 19 и 20 веков (частые и редкие)
- График динамики архаизмов по десятилетиям (1720-2000)
- Сравнение авторов по использованию архаизмов
- Сравнение по векам
- Таблицы со статистикой (CSV)

**Ключевые находки:**
- Пик использования: 1720s (27.69 архаизмов на 1000 слов)
- Минимум: 1990s (5.92 архаизмов на 1000 слов)
- Лидер среди авторов: Михаил Лермонтов (12.81 на 1000 слов)

---

### 4. Сравнительный анализ ✅
- Сравнение использования архаизмов у разных авторов
- Выявление периодов интенсивной архаизации
- Анализ корреляции с литературными течениями

**Скрипты:**
- `src/comparative_analysis.py` - полный сравнительный анализ

**Результаты:**
- График сравнения литературных течений
- Временная шкала с выделением течений
- Таблица по течениям (CSV)
- **Аналитический отчёт** `ANALYSIS_REPORT.txt`

**Ключевые выводы:**
- Классицизм: max архаизация (16.93 на 1000 слов)
- Советская поэзия: min архаизация (6.62 на 1000 слов)
- Разброс между течениями: σ = 3.55
- **Циклический паттерн:** Рост архаизмов к концу каждого раннего периода 
  (Классицизм, Сентиментализм, Романтизм, Реализм). Интерпретация: 
  "прощание с эпохой" через обращение к традиции
- **Вывод:** Литературное течение - значимый фактор использования архаизмов

---

### 5. Визуализация и выводы ✅
- Создание итоговых графиков и таблиц
- Формулирование выводов
- Подготовка краткого отчёта

**Статус:** Выполнено. Все графики, таблицы и аналитический отчёт созданы.

## Быстрый старт

### Установка зависимостей
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Запуск по этапам

**Этап 1: Подготовка данных**
```bash
python src/parse_xml_to_csv.py      # Создаёт poems.csv и poems.parquet
python src/example_load_data.py     # Проверяет загрузку данных
```

**Этап 2: Словарь архаизмов**
```bash
python src/parse_archaisms.py           # Создаёт archaisms.csv и archaisms.parquet
python src/example_find_archaisms.py    # Ищет архаизмы в стихах
```

**Этап 3: Анализ частотности**
```bash
python src/word_frequency_analysis.py   # Создаёт облака слов в results/
```

## Основные результаты исследования

### Количественные данные:
- ✅ Проанализировано **1,726,105 слов** в 16,694 стихотворениях
- ✅ Найдено **14,922 архаизма** (средняя частотность: 8.64 на 1000 слов)
- ✅ Изучено **8 литературных течений** и **195 авторов**

### Ключевые открытия:

1. **Историческая динамика:**
   - Снижение архаизации на **78.6%** за 270 лет (1720-1990)
   - Пик: 1720s (27.69 на 1000) → Минимум: 1990s (5.92 на 1000)

2. **Циклический паттерн (новая находка!):**
   - Рост архаизмов **к концу каждого раннего периода**:
     - Конец Классицизма, Сентиментализма, Романтизма, Реализма
   - Интерпретация: "прощание с эпохой" через обращение к традиции
   - С модернистскими течениями паттерн исчезает

3. **Влияние литературных течений:**
   - Классицизм: 16.93 на 1000 слов (максимум)
   - Советская поэзия: 6.62 на 1000 слов (минимум)
   - Разброс σ = 3.55 (статистически значим)

4. **Лидеры среди авторов:**
   - Михаил Лермонтов: 12.81 на 1000 слов
   - Федор Тютчев: 12.44
   - Александр Пушкин: 11.42
