# Словарь архаизмов и устаревших слов

## Источник данных

Словарь составлен на основе материалов сайта [Азбука веры](https://azbyka.ru/otechnik/Spravochniki/slovar-ustarevshih-slov/)

Оригинальный файл: `устаревшие.txt`

## Структура файлов

### 1. `archaisms.csv` / `archaisms.parquet`
**Полный структурированный словарь с определениями**

Колонки:
- `word` - основное слово (архаизм)
- `definition` - определение слова
- `variants` - варианты написания (если есть)
- `letter` - буква алфавита (для группировки)
- `original` - оригинальная форма из источника

**Примеры:**
```csv
word,definition,variants,letter,original
Абие,тотчас, с тех пор как, когда.,,А,Абие
Аз,местоимение «я» или наименование первой буквы алфавита.,,А,Аз
Аз,названия первых букв славянского алфавита.,"буки, веди",А,"Аз, буки, веди"
```

### 2. `archaisms_wordlist.txt`
**Простой список слов для быстрого поиска**

Формат: одно слово на строку (все в нижнем регистре)

```
абие
абы
агнец
аз
...
```

## Категории устаревших слов

Согласно источнику, устаревшие слова делятся на две группы:

### Архаизмы
Слова, вышедшие из употребления, но имеющие синонимы в современном языке:
- **десница** – правая рука
- **ланиты** – щёки
- **рамена** – плечи
- **чресла** – поясница

### Историзмы
Слова, обозначающие исчезнувшие явления и предметы:
- **редут, кольчуга, забрало, пищаль** (военная тематика)
- **просак, светец, ендова, камзол, армяк** (быт и одежда)
- **царь, боярин, стольник, конюший, бурлак** (сословия и должности)

## Использование

### Загрузка словаря

**Python (pandas):**
```python
import pandas as pd

# Полный словарь с определениями
df = pd.read_parquet('dataset/archaisms.parquet')

# Простой список слов
with open('dataset/archaisms_wordlist.txt', 'r', encoding='utf-8') as f:
    archaisms = set(line.strip().lower() for line in f)
```

### Поиск архаизмов в тексте

```python
import re

def find_archaisms(text, archaisms_set):
    """Находит архаизмы в тексте."""
    words = re.findall(r'\b[а-яёА-ЯЁ]+\b', text.lower())
    return [word for word in words if word in archaisms_set]

# Пример
text = "Сей муж имел десницу крепкую и очи зоркие."
found = find_archaisms(text, archaisms)
# Результат: ['сей', 'десницу', 'очи']
```

## Ограничения и улучшения

### Текущие ограничения:
1. **Нет морфологии** - слова только в базовой форме
2. **Простой поиск** - не учитывает склонения и спряжения
3. **Смешанные категории** - нет явного разделения на архаизмы и историзмы

### Возможные улучшения:
1. **Лемматизация** - использовать pymorphy2 для приведения к начальной форме
2. **Категоризация** - разделить на архаизмы, историзмы, старославянизмы
3. **Расширение** - добавить морфологические формы
4. **Контекст** - учитывать контекст использования слова
